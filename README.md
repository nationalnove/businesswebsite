# Team Portfolio Website

This is a simple static website for a five-person professional team with skills in:

- SOC / Cybersecurity
- Digital Advertisement
- Content Creation
- Technical Support
- US Medical Billing

## Files included

- `index.html` - main website content
- `styles.css` - website styling
- `README.md` - setup guide

## How to edit

Open `index.html` and replace the placeholder details:

- `Your Team Name`
- `hello@yourteamname.com`
- Member names
- Member summaries
- Services or skills you want to add/remove

## How to publish using GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `README.md` to the repository.
3. Go to the repository `Settings`.
4. Go to `Pages`.
5. Under `Build and deployment`, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Click `Save`.
7. GitHub will provide your website link after publishing.

Your website URL will usually look like this:

`https://yourusername.github.io/repository-name/`

## Safety note

This website intentionally uses an email link instead of a long contact form. It does not collect passwords, login credentials, medical records, financial details, or government IDs.
